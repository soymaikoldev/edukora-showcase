# EduKora

EduKora is a gamified learning platform designed to help people understand Solana and practice real onchain actions in a simple, guided way.

## Problem

For many new users, blockchain education is still too fragmented, too technical, and disconnected from real action. Learning about wallets, transactions, staking, swaps, or onchain tools often feels overwhelming.

## Solution

EduKora makes onboarding easier through structured learning paths, gamified progress, practical exercises, and real-world blockchain interactions.

## What users can do

- Learn the basics of Solana step by step
- Understand wallets, seed phrases, and onchain safety
- Practice actions like connecting a wallet and signing transactions
- Progress through interactive learning routes
- Earn XP, rewards, and recognition for completing milestones

## Current status

EduKora is currently in private development / early beta.

- Core product architecture in progress
- Documentation in active development
- Initial testers already reviewing the experience
